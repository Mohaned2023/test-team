## About The Project:
This project aims to test the team's capabilities.

In terms of frontend and backend, as well as dividing\
roles and working as a team.

## What will we use?
- Backend: `Nest.js`
- Frontend: `React.js`
- Database: `Postgres`
- Installation: `Docker`
- Server: `Nginx`
