## Create issue
1. Go to main repo of the project.
2. click issue and create it.
3. Enter this formate:
```md
Title: 404 server error.
description: 
    # What happened?
        ...
    # How to create the error or the issue?
        1. 
        2. 
        3. 
        ...
    # Expected behavior without the error or the issue?
        ...
```
4. Click Send.
